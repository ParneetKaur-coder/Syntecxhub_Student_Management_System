# 🎓 Student Management System

> A **Java Console-Based Student Management System** developed during my internship at **SyntecxHub**. This project demonstrates the practical implementation of **Core Java**, **Object-Oriented Programming (OOP)**, **ArrayList**, **File Handling**, and **Exception Handling** by building a complete CRUD-based application.

---

## 📖 Project Overview

The **Student Management System** is a menu-driven console application that allows users to manage student records efficiently. It provides functionalities to add, view, search, update, and delete student information while storing data permanently using file handling.

This project was developed to strengthen my understanding of Java fundamentals and to gain practical experience in designing real-world applications using object-oriented principles.

---

## ✨ Features

* ➕ Add New Student Records
* 📋 View All Students
* 🔍 Search Student by Student ID
* ✏️ Update Student Information
* ❌ Delete Student Records
* 💾 Store Data Using File Handling
* 📂 Automatically Load Saved Data
* ✅ Input Validation
* ⚠️ Exception Handling
* 🖥️ Interactive Menu-Driven Console Interface
* 📊 Clean and Formatted Console Output
